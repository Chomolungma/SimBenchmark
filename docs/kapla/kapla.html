Kapla test

Test scenario

The setup of the test is as follows:
  • static ground
  • stack of 102 blocks
  • Metric: time until the stack collapses

Tested solver list

  • Rai
    • Bisection method
  • Bullet
    • Sequential Impulse
  • ODE
    • Dantzig[1]
    • LCP PGS
  • MuJoCo
    • PGS
    • CG
    • Newton
  • Dart
    • Dantzig
    • LCP PGS

Commentary

  • [1] ODE Dantzig solver is significantly slow with large number of contacts thus was excluded in this test.
