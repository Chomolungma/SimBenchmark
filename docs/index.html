SimBenchmark

SimBenchmark is a project for benchmark of rigid body simulators with contacts.

List of Simulators

The list of the simulators is as follows:

  • RaiSim (unreleased)
    • What is RaiSim?
  • Bullet Physics
  • Open Dynamics Engine (ODE)
  • Multi-Joint dynamics with Contact (MuJoCo)
  • Dynamic Animation and Robotics Toolkit (DART Sim)
Read this page for more details.

Methodology

In this benchmark, we measure accuracy and CPU time of simulation with different time step. Since accuracy of numerical solution is usually better when the time step is smaller, we considered this speed-accuracy trade off in evaluation. This evaluation method was proposed by Tom Erez et al.

For accuracy metric, we use squared velocity error of simulation when an analytical solution of trajectory is applicable. Besides, for complex systems that analytical solution of trajectory is not available, we investigate penetration error, momentum or energy.

More details of benchmark methodology is here.

Test results

  • Rolling test: friction model test
  • Bouncing test: single-body elastic collision test
  • 666 balls test: single-body hard contact test
  • Elastic 666 balls test: single-body energy test
  • Tower: single-body stacking test
  • ANYmal PD control test: articulated-robot-system speed test
  • ANYmal momentum test: articulated-robot-system momentum test
  • ANYmal energy test: articulated-robot-system energy test
