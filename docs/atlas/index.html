Atlas PD control test

Atlas test is to test computational efficiency in solving articulated dynamics. In this test, a 35 DOF bipedal robot Atlas is feedback-controlled to lie on a flat ground with 4 contacts.

The test focuses on:

  1. Computational cost
  2. Scalability

Test scenario

Tested solver list

  • Bullet

  • ODE1)

  • MuJoCo
    • PGS
    • CG
    • Newton
  • Dart
    • Dantzig
    • LCP PGS

Commentary

  1. We excluded ODE for this test since it is too slow to simulate such a complex robot system. See this page to check its speed of articulated system simulation and scalability.
